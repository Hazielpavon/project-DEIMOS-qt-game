# Proyecto Deimos - 2D Qt Game

Este repositorio contiene un videojuego 2D desarrollado en C++ utilizando Qt como parte de mi proceso de aprendizaje en la universidad.

El objetivo principal de mantener este repositorio publico es demostrar que a pesar de ser un grupo de Tegucigalpa y San Pedro, la distancia no nos detuvo de entregar un buen trabajo y fue mi primera vez trabajando por zoom y github de forma intensiva. 

## Proposito del repositorio

Este proyecto forma parte de nuestro crecimiento academico y personal como estudiantes de Ingenieria en Sistemas. A traves de este repositorio buscamos evidenciar la evolucion de nuestros conocimientos, la practica constante y la mejora progresiva en el desarrollo de software.

## Descripcion del proyecto

Proyecto Deimos es un videojuego 2D donde el jugador avanza por diferentes niveles enfrentando enemigos, utilizando habilidades y superando desafios hasta llegar a un jefe final.

El juego incluye mecanicas de movimiento, ataques, enemigos, rondas especiales y una estructura basada en objetos para organizar los elementos principales del juego.
